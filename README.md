# TP-Decrypt
TP-Link Router bin decryption tools

<h3 align="left">INSTALL :</h3>

````bash
pkg update && upgrade -y
pkg install python -y
pkg install git -y
pkg install ruby -y
gem install lolcat -y
git clone https://github.com/ShTasrif/TP-Decrypt
````

## How to use 
<h3 align="left">RUN :</h3>

````bash
cd TP-Decrypt
python main.py
````
#### Run :

````bash
tpdec
````
